Classificando copas de árvores

Introdução

Mapear as espécies de árvores em uma vasta floresta é uma tarefa difícil, parte da complicação vem do fato da falta de acesso a dados anotados, em outras palavras, a proporção de árvores das quais sabemos as espécies e as quais não sabemos é bem baixa. Com isso em mente, no decorrer desse trabalho será proposto uma solução para a problemática utilizando algoritmo de pseudo-labeling juntamente com reinforcement learning.

Predições: pseudo-label

pseudo-labeling

O algoritmo de pseudo-labeling consiste em alimentar o conjunto de treinamento com labels virtuais. Para isso, utiliza-se dos poucos dados anotados disponíveis no dataset para treinar um classificador, e com base nessas classificações feitas podemos decidir incluir ou não novas amostras no conjunto de treino. Uma forma que segundo Cascante-Bonila (2020) acompanha o estado da arte é utilizar de análise de currículo juntamente com o pseudo-labeling, que consiste em avaliar apenas os pontos com maior probabilidade de ser de determinada classe.

O gráfico acima representa o algoritmo de pseudo-labeling com análise de currículo classificando 900 pontos tendo acesso inicialmente a apenas 100. Além de utilizar da análise de currículo, queremos dar um passo além e incluir um algoritmo de RL que tome a decisão de incluir, ou não, os novos pontos. Dessa forma conseguimos adicionar informações adicionais na hora da decisão, como por exemplo a proporção da distribuição das árvores na floresta.

Dados reais

sampling

Em La Rosa (2021), foi implementado uma rede neural convolucional para tratar de dados hiperespectrais de florestas brasileiras, tais dados possuem alto peso em memória e exige vasto poder computacional para tratá-los. Para ultrapassar essa dificuldade de hardware e de tempo, os dados foram substituídos por dados gerados partindo de uma mistura de gaussianas.
